# Terminal and Shell Configuration

**Primary Shell**: Fish
**Primary Terminal**: Kitty
**Configuration Paths**: `~/.config/fish/`, `~/.config/kitty/`

---

## Fish Shell Configuration

### Overview

Fish shell uses a modular configuration system with numbered files in `conf.d/` for initialization order.

### Plugin Manager & Plugins

**Plugin Manager**: fisher

**Installed Plugins:**
1. `jorgebucaran/fisher` - Plugin manager
2. `jorgebucaran/hydro` - Fast prompt system
3. `kidonng/zoxide.fish` - Smart directory jumping
4. `patrickf1/fzf.fish` - Fuzzy finder integration
5. `jorgebucaran/nvm.fish` - Node version manager

### Configuration Structure

```
~/.config/fish/
├── config.fish              # Main config (minimal)
├── fish_plugins             # Plugin list
├── fish_variables           # Universal variables
├── conf.d/                  # Modular config files
│   ├── 10_core.fish         # Core environment
│   ├── 20_tools.fish        # Tool configuration
│   ├── 25_prompt.fish       # Custom prompt
│   └── 30_abbreviations.fish # Command abbreviations
├── functions/               # Custom functions
└── completions/             # Tab completions
```

### Core Environment (10_core.fish)

**Environment Variables:**
```fish
set -gx EDITOR vim
set -gx VISUAL vim
set -gx TERM xterm-256color
set -gx LANG en_US.UTF-8
```

**XDG Directories:**
```fish
set -gx XDG_CONFIG_HOME ~/.config
set -gx XDG_CACHE_HOME ~/.cache
set -gx XDG_DATA_HOME ~/.local/share
```

**Modern Tool Aliases:**
- `bat` → `cat` (syntax-highlighted)
- `eza`/`exa` → `ls`, `la`, `ll`
- `rg` → `grep` (ripgrep)
- `fd` → `find`

**PATH Additions:**
- `~/.local/bin`
- `~/.cargo/bin`
- `~/.krew/bin`

### Tool Configuration (20_tools.fish)

**Hydro Prompt:**
```fish
set -g HYDRO_SYMBOL_PROMPT ❯
set -g HYDRO_COLOR_ACCENT blue
set -g HYDRO_COLOR_STATUS_SUCCESS green
set -g HYDRO_COLOR_STATUS_ERROR red
```

**fzf Configuration (Tokyo Night theme):**
- Uses `fd` for file discovery
- Ctrl+T: File search
- Alt+C: Directory search

**Integrations:**
- zoxide (smart cd)
- direnv (project environments)
- nvm (Node version management)
- Rust environment

## Kitty Terminal Configuration

**Configuration Path**: `~/.config/kitty/`

### Main Configuration (kitty.conf)

```conf
# Font Configuration
font_family       Iosevka Term
bold_font         auto
italic_font       auto
bold_italic_font  auto
font_size         12.0

# Window Layout
window_padding_width  10

# Colors
include colors.conf
```

## Bash Configuration (Minimal)

**File**: `~/.bashrc`

```bash
alias ls='ls --color=auto'
alias grep='grep --color=auto'
PS1='[\u@\h \W]\$ '
```

Fish is the primary shell; Bash is minimal fallback.
