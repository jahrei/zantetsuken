## Overview

This Hyprland setup features:
- **i3-like tiling** with the hy3 plugin for advanced window management
- **Material You color scheme** generated by Matugen
- **Multi-monitor support** with 3 connected displays
- **Custom workspace layouts** for different use cases
- **Advanced keybindings** with vim-style navigation
- **RGB synchronization** with hardware (OpenRGB)
- **Custom scripts** for screenshots, layout management, and window control

## Monitor Configuration

### Connected Monitors

```conf
monitor = DP-2,      2560x1440@60,   -2560x0, 1      # Left monitor (60Hz)
monitor = DP-1,      2560x1440@144,  0x0,     1      # Primary monitor (144Hz)
monitor = Unknown-2, 1920x1080@60,   320x1440, 1     # Below DP-1 (1080p)
```

**Layout:**
- **DP-1** (Primary): 2560x1440 at 144Hz, center position
- **DP-2** (Left): 2560x1440 at 60Hz, to the left of primary
- **Unknown-2** (Bottom): 1920x1080 at 60Hz, below primary

## Plugins

### hy3 Plugin (i3-like Tiling)

**Installation**: Via hyprpm (Hyprland Plugin Manager)

```bash
hyprpm add https://github.com/outfoxxed/hy3
hyprpm enable hy3
hyprpm reload
```

**Auto-rebuild**: Pacman hook at `/etc/pacman.d/hooks/hyprpm.hook` automatically rebuilds hy3 when Hyprland updates.

**Features:**
- i3-style window tiling
- Tab group management (`Super+W`)
- Vertical/horizontal split containers (`Super+V`)
- Advanced focus management

## Visual Settings

### General Settings

```conf
general {
    gaps_in = 5              # Gap between windows
    gaps_out = 20            # Gap around screen edges
    border_size = 2          # Window border thickness
    col.active_border = $primary
    col.inactive_border = $outline_variant
    layout = hy3             # Use hy3 layout engine
}
```

### Decorations

```conf
decoration {
    rounding = 10            # 10px corner radius
    active_opacity = 1.0
    inactive_opacity = 1.0

    shadow {
```

## Keybindings

### Basic Window Management

| Keybind | Action |
|---------|--------|
| `Super + Return` | Open terminal (kitty) |
| `Super + Q` | Close active window |
| `Super + M` | Exit Hyprland |
| `Super + E` | Open file manager (dolphin) |
| `Super + Shift + Space` | Toggle floating mode |
| `Super + D` | Open application menu (rofi) |
| `Super + X` | Open script menu |

### Movement & Navigation (hy3 i3-style)

| Keybind | Action |
|---------|--------|
| `Super + H/J/K/L` | Move focus (vim keys) |
| `Super + Shift + H/J/K/L` | Move window (vim keys) |
| `Super + Arrow Keys` | Move focus |
| `Super + Shift + Arrow Keys` | Move window |

### Workspace Management

| Keybind | Action |
|---------|--------|
| `Super + 1-9` | Switch to workspace 1-9 |
| `Super + 0` | Switch to workspace 10 |
| `Super + Shift + 1-9` | Move window to workspace |

### i3-like Container Management (hy3)

| Keybind | Action |
|---------|--------|
| `Super + V` | Create vertical split group |
| `Super + W` | Create tabbed group |
| `Super + S` | Toggle stacking/tabbed mode |
| `Super + A/Z` | Focus prev/next tab in group |

### Layout Loading

| Keybind | Action |
|---------|--------|
| `Super + Ctrl + L` | Load default layout |
| `Super + Ctrl + Shift + D` | Load dev layout |
| `Super + Ctrl + Shift + P` | Load productivity layout |

### Screenshot Controls

| Keybind | Action |
|---------|--------|
| `Ctrl + Home` | Screenshot to clipboard |
| `Ctrl + Shift + Home` | Screenshot, save, and copy |
| `Ctrl + Alt + Home` | Screenshot with OCR |

### Multimedia Keys

| Keybind | Action |
|---------|--------|
| `XF86AudioRaiseVolume` | Volume up 5% |
| `XF86AudioLowerVolume` | Volume down 5% |
| `XF86AudioMute` | Toggle mute |
| `XF86AudioPlay/Pause` | Play/pause media |

## Layout System

### Saved Layouts

Located in `~/.config/hypr/layouts/`:

- **default.layout** - Basic terminal + browser setup
- **dev.layout** - IDE, terminal, browser in tabs
- **productivity.layout** - Multi-window productivity setup

### Creating Custom Layouts

```bash
# Save current workspace
~/.config/hypr/scripts/save-layout.sh my-layout

# Load a layout
~/.config/hypr/scripts/load-layout.sh my-layout
```

## Color Scheme

### Material You Theme

Generated by Matugen from wallpaper. Colors stored in `~/.config/hypr/colors.conf`.

**Primary Colors:**

| Variable | Value | Purpose |
|----------|-------|---------|
| `$primary` | #abc7ff | Active borders, accents |
| `$secondary` | #bec6dc | Secondary elements |
| `$tertiary` | #7dd3fc | Tertiary accents |
| `$surface` | #111318 | Background |
| `$on_surface` | #e2e2e9 | Text color |
| `$error` | #ffb4ab | Error states |

### Applying New Theme

```bash
matugen image /path/to/wallpaper.png
```

This automatically updates colors across Hyprland, Waybar, Kitty, and sets the wallpaper via swww.
